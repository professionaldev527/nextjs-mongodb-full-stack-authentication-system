# Next.js & MongoDB Full-Stack Authentication System 🔐

A robust, full-stack authentication system built with **Next.js**, **MongoDB**, and **TypeScript**. This project serves as a complete boilerplate for implementing secure user registration, login, and protected routes in a modern web application, utilizing best practices for database integration and state management.

## 🚀 Features

- **User Authentication:** Secure Sign-Up and Log-In functionality.
- **Session Management:** Secure token-based authentication (JWT) with HTTP-only cookies.
- **Password Security:** Passwords are mathematically hashed and salted before database storage (e.g., bcrypt).
- **Protected Routes:** Middleware implementation to restrict access to authenticated users only.
- **Database Integration:** Direct and efficient connection to MongoDB using Mongoose.
- **Type Safety:** Entirely built with TypeScript for better developer experience and fewer runtime errors.
- **Responsive UI:** Styled utility-first with Tailwind CSS for a clean, modern interface.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (React Framework)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Database:** [MongoDB](https://www.mongodb.com/) (with Mongoose ORM)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Authentication:** Custom JWT / Cookie-based Auth

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

Make sure you have the following installed:
- [Node.js](https://nodejs.org/en/) (v16.x or later)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)
