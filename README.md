# Full-Stack Authentication & Management System

A robust full-stack web application built with **Next.js 14**, **MongoDB**, and **Clerk**. This project demonstrates modern authentication flows, database integration, and a sleek user interface for task management.

## 🚀 Live Demo

You can access the live application using the links below:

*   **Primary Domain:** [todo-master.theprofdev.com](https://todo-master.theprofdev.com)
*   **Vercel Deployment:** [Live Preview](https://next-js-todo-master-clerk-neon-db-postgre-9ccio5bgl.vercel.app)

---

## ✨ Features

-   **Authentication:** Secure login, sign-up, and social auth via Clerk.
-   **Full-Stack CRUD:** Create, Read, Update, and Delete functionality powered by Next.js Server Actions.
-   **Database:** Scalable data storage using MongoDB.
-   **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop views.
-   **Middleware:** Protected routes and session management.

## 🛠️ Tech Stack

-   **Framework:** [Next.js 14](https://nextjs.org/) (App Router)
-   **Authentication:** [Clerk](https://clerk.com/)
-   **Database:** [MongoDB](https://www.mongodb.com/)
-   **Styling:** [Tailwind CSS](https://tailwindcss.com/)
-   **Deployment:** [Vercel](https://vercel.com/)

---

## ⚙️ Local Setup

Follow these steps to run the project locally:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/professionaldev527/nextjs-mongodb-full-stack-authentication-system.git](https://github.com/professionaldev527/nextjs-mongodb-full-stack-authentication-system.git)
    cd nextjs-mongodb-full-stack-authentication-system
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Environment Variables:**
    Create a `.env.local` file in the root directory and add your credentials:
    ```env
    NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=your_key
    CLERK_SECRET_KEY=your_key
    MONGODB_URI=your_mongodb_uri
    ```

4.  **Run the development server:**
    ```bash
    npm run dev
    ```
